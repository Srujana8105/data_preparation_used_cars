# Data Preparation for Machine Learning

This project focuses on preparing a used car dataset for machine learning.

## What I did

* Loaded the dataset and checked its structure
* Handled missing values using mean or median depending on the data
* Removed duplicate rows
* Converted categorical data into numerical format using one-hot encoding
* Split the data into training and testing sets (80/20)
* Applied feature scaling using StandardScaler

## Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Final Output

The dataset is now clean and ready to be used for building a regression model.
